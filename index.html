Sublime textda Githubni ishga tushirish

Sublime-Textda yozgan kodimizni GitHubga joylash buyruqlari:

  1. CTRL+SHIFT+P = install package Git paketini o`rnatamiz (bu paketni urnatishdan avval kompiyuterimizga Gitbush va Sublime Merge)dasturlarini urnatib olishimiz kerak, albatta:
  2. CTRL+SHIFT+P = install package sublime GitHub paketini o`rnatamiz
  • CTRL+SHIFT+P = Git + Enter
  • CTRL+SHIFT+P = Git add all
  • CTRL+SHIFT+P = Git commid yangi oyna ochiladi va comentariya yozamiz (githubga nimani yuklayotganimizni bildiradigan bironta nom)M:initial commit X bosib chiqib ketamiz
  • CTRL+SHIFT+P = Git custom yangi oyna ochiladi. Ochilgan oynaga(githubda yangi repozistory ochib uni linkini) Push(......link)masterjoylaymiz.
